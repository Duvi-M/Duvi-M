<h2>🔬 Research & Thesis</h2>

<h3>DULoRA — Dynamic Utility-based LoRA Rank Allocation</h3>

<p>
My master's thesis focuses on improving <b>Low-Rank Adaptation</b> for transformer fine-tuning.
Standard LoRA usually applies the same rank across all target layers, even though different layers may contribute differently to the final task.
</p>

<p>
In this project, I study an adaptive approach where LoRA ranks are assigned based on layer utility signals, with the goal of using the trainable parameter budget more efficiently.
</p>

<p>
Main research and engineering components:
</p>

<ul>
  <li>Adaptive rank allocation for LoRA layers</li>
  <li>Gradient-based layer utility estimation</li>
  <li>PyTorch and Hugging Face Transformers implementation</li>
  <li>PEFT-compatible fine-tuning pipeline</li>
  <li>Evaluation on NLP text classification tasks</li>
  <li>Comparison against standard fixed-rank LoRA</li>
</ul>
